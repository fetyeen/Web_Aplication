# University Course Enrollment Web System

## 📌 Overview

This project is a **web-based course enrollment system** designed for higher education institutions. Built entirely with HTML, CSS, and JavaScript, it allows students to register for courses, view their schedule, and manage personal data — all without using a backend or external database.

## 🛠 Technologies Used

- **HTML** – Structure of web pages and user interface.
- **CSS** – Styling, layout, and responsive design.
- **JavaScript** – Application logic and interaction with `localStorage`.

> The entire system operates client-side using browser storage.

## 👥 User Roles

### 👤 Student
- Register and log in to the system.
- Browse available courses, lecturers, and dates.
- Enroll in a course (choose lecturer and date).
- View current and completed courses.
- Access and edit profile information.

### 👨‍💼 Administrator
- Secure login with credentials (`Admin` / `1234`).
- Add/remove:
  - Courses
  - Lecturers
  - Dates
- View all students and their registrations.
- Use a dedicated admin panel separated from the student view.

## 📂 How to Run

1. Clone or download the repository as a ZIP:
   [GitHub Repository](https://github.com/fetyeen/Web_Aplication)
2. Extract the ZIP file.
3. Open `log.html` in a web browser.
4. To access the admin panel:
   - **Username:** Admin  
   - **Password:** 1234
